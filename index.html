Communicating Data-driven Policy Research

A proposal by Isabel Meirelles, Botond Reszegh and Csongor Bartus

all-in-one solution / A website which prints.

We have learned with great interest about the call made public by the OSI and we propose an all-in-one solution, a webpage to print, which believe will be the most optimal to achieve the desired outcomes.

We chose this solution because the need to produce both print and online tools for data visualization and we believe that the time we can save on preparing these tools would be most valuable for our intended partners.

Our solution is unique as it proposes to allow you to freely add and manage projects.

Deliverables and Plans

  1. Beautiful design
  2. Better user experience
  3. State of art infographics & technology

three variants / Choose yours.

#1 — First and most important we have found the design of the showcased projects are below international standards. Your message is not presented in the most pleasant way to readers. That's where we can help imediately: with a fresh look, a fresh design which delivers the message.

#2 — Once the design is ready and the message is delivered the next step would be to enable people to play more with the information presented. And come back later to check out new things, pieces of knowledge. This requires to add a new, more interactive and usable layer to your existing solution.

#3 — Finally, if there is a desire to reach highest standards we can consult our expert and re-design all infographics which are currently obsolete. In many cases wrong techniques are applied to good data.

Technologies

bleeding edge / Open Source

Open web technologies

Open web technologies

We consider ourselves hackers: walking on, and pushing the limits of technology. We've grown up in closed systems now we value openness. In this spirit everything we create is built on open source.

Valid HTML5, CSS3, Javscript with SASS and Compass frameworks for cross browser compatibility. SVG, CSS3D, WebGL for animations and rich interactions. D3.js and Processing.js for interactive visualizations.

jsPDF, CSS3 Page Media Module for printing a website instead of closed source alternatives like Prince or Antenna House. You name it — we find an open source solution and integrate into your project.

Responsive web design

Responsive web design

Smartphones and tablets are first order citizens of the web like desktop monitors. We are always designing for them.

Additionally, making a website to print uses this same technology. Print is considered another `device` the website must adapt.

We've started creating responsive web sites back in 2010 being one of the early birds. Now we move forward from screen to paper and maybe more beyond.

Github: code, deliverables and hosting

We use Github from the beginning. It keeps our assets safe and collaboration easy. This current site is fully Github driven. All the source code and deliverables you can find in the repository.

A framework for visualizations

A framework for visualizations

We are planning to collect in a repository the best visualization tools with a special focus on open source. You can use them freely in your future projects. It would be like a library where you can easily choose a technology and apply to your needs.

There are many hidden gems on the net. For example the timeline for project variants you've seen before is created through Google Docs. You fill out a form and get a link you insert into your webpage. And that's all.

We think such tools will help you work better and faster in the future with your projects.
