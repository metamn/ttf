Communicating Data-driven Policy Research

A proposal by Isabel Meirelles, Botond Reszegh and Csongor Bartus

all-in-one solution / A website which prints.

We have learned with great interest about the call made public by the OSI and we propose an all-in-one solution, a webpage to print, which believe will be the most optimal to achieve the desired outcomes.

We chose this solution because the need to produce both print and online tools for data visualization and we believe that the time we can save on preparing these tools would be most valuable for our intended partners.

Our solution is unique as it proposes to allow you to freely add and manage projects.

Deliverables and Plans

  1. Beautiful design
  2. Better user experience
  3. State of art infographics & technology

three variants / Choose yours.

First and most important we have found the design of the showcased projects are below international standards. Your message is not presented in the most pleasant way to readers. That's where we can help imediately: with a fresh look, a fresh design which delivers the message.

Once the design is ready and the message is delivered the next step would be to enable people to play more with the information presented. And come back later to check out new things, pieces of knowledge. This requires to add a new, more interactive and usable layer to your existing solution.

Finally, if there is a desire to reach highest standards we can consult our expert and re-design all infographics which are currently obsolete. In many cases wrong techniques are applied to good data.

Technologies

bleeding edge / Open Source

HTML5, CSS3, Javascript

HTML5, CSS3, Javascript

Almost every old technology like Flash, Java applets are replaced with open source counterparts.

For in-browser animations we have CSS3D, WebGL and SVG as native technologies, and Javascript libraries like D3.js and Processing.js

We exclusively use them when creating web sites and applications. Like in this proposal.

Responsive web design

Responsive web design

Smartphones and tablets are first order citizens of the web like desktop monitors. We are always designing for them.

To make cross-browser compatible we use open source frameworks to create responsive stylesheets like SASS and Compass.

Visualization framework

Visualization framework

We are planning to collect in a repository the best visualization tools with a special focus on open source.

Then you can use these in your future projects. It would be like a library where you can easily choose a technology and apply to your needs.

For example to present the timeline for each variant we use Timeline.js, a tool which takes information from Google Docs and presents on the web via an iframe.
